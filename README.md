# Mini-hacker
Mini Hack – الهاكر الصغير
An educational interactive hacking simulation game built using HTML, CSS, and JavaScript. Designed as a terminal-style experience with six levels to simplify basic cybersecurity concepts in a fun and engaging way.

Features
•	Bilingual: Arabic / English
•	Realistic terminal-style interface
•	Six difficulty levels
•	Score saving and results display

How to Run
Open index.html in any web browser.

Tools Used
HTML • CSS • JavaScript

Project Goal
Teach simple cybersecurity concepts through an enjoyable, interactive experience.
